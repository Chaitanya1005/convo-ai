# Rupeezy AI Sales Agent

## 🚀 Overview

Rupeezy AI Sales Agent is a full-stack AI-powered system designed to automate lead conversations and improve conversion rates for brokerage services.

It simulates a human-like sales agent that communicates in natural Hindi, understands user intent, and responds intelligently.

---

## 🧠 Features

* 🗣️ Natural Hindi Conversation
* 🎯 Lead Personalization (Name, City, Interest)
* 📈 Conversion-focused dialogue
* ⚡ Real-time AI responses
* 🔗 Full-stack integration (Frontend + Backend)

---

## 🏗️ Architecture

User → Frontend (Next.js) → Backend (Django) → AI Model → Response

---

## ⚙️ Tech Stack

* Frontend: Next.js, TypeScript, Tailwind CSS
* Backend: Django, Django REST Framework
* AI: OpenAI / Groq API
* Tools: Python, REST APIs

---

## 📂 Project Structure

* `/convo-ai` → Frontend UI
* `/rupeezy` → Backend + AI logic

---

## ▶️ How to Run

### Backend

```bash
cd rupeezy/rupeezy_ai
pip install -r requirements.txt
python manage.py runserver
```

### Frontend

```bash
cd convo-ai
npm install
npm run dev
```
